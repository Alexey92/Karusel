Установка и запуск на Windows (локальный сервер)

1. Установить Python 3.12

Скачать с python.org. При установке — галочка «Add Python to PATH».

2. Установить зависимости

cd C:\karusel\server
python -m pip install -r requirements.txt

3. Запустить

python main.py

4. Проверить

Браузер: http://localhost:5050/

5. Автозапуск

Создать ярлык для start_server.bat в папке shell:startup.

Установка и запуск на macOS (центральный сервер)

1. Установить Python 3.12

Скачать с python.org, запустить .pkg.

2. Установить PostgreSQL

Скачать Postgres.app, запустить, нажать Initialize.

echo 'export PATH="/Applications/Postgres.app/Contents/Versions/latest/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
createdb karusel
psql -d karusel -c "CREATE USER karusel WITH PASSWORD 'karusel'; GRANT ALL PRIVILEGES ON DATABASE karusel TO karusel;"
psql -d karusel -c "GRANT ALL ON SCHEMA public TO karusel;"

3. Установить Git (опционально)

xcode-select --install

4. Скопировать проект

Через флешку в ~/Documents/karusel или через Git.

5. Установить зависимости

cd ~/Documents/karusel/cloud
python3.12 -m pip install -r requirements.txt

6. Запустить

python3.12 main.py

7. Проверить

Браузер: http://localhost:5050/admin (admin / admin123).

8. Автозапуск (launchd)

nano ~/Library/LaunchAgents/com.karusel.cloud.plist

Вставить XML (см. ниже), заменить пути.

launchctl load ~/Library/LaunchAgents/com.karusel.cloud.plist

Добавление нового адреса (объекта)

Шаг 1: Создать адрес в облачной админке

http://IP:5050/admin → вкладка «Адреса» → «Добавить адрес».

Шаг 2: Получить api_key

psql -d karusel -c "SELECT id, name, api_key FROM locations;"

Шаг 3: Сгенерировать ключ вручную (если нужно)

python3.12 -c "import secrets; print(secrets.token_urlsafe(32))"

Шаг 4: Добавить автоматы

В админке: «Адреса» → 🔧 → добавить автоматы. Или SQL:

INSERT INTO machines (name, location_id) VALUES ('Автомат №1', 2), ... ('Автомат №10', 2);

Шаг 5: Настроить мини-ПК на объекте

В start_server.bat:

set KARUSEL_CLOUD_URL=http://IP_ЦЕНТРАЛЬНОГО_СЕРВЕРА:5050/api/event
set KARUSEL_CLOUD_KEY=api_key_из_пункта_2
set KARUSEL_LOCATION_ID=id_адреса

Шаг 6: Проверить связь

Invoke-RestMethod -Uri http://IP_ЦЕНТРАЛЬНОГО_СЕРВЕРА:5050/ -Method Get

Работа с PostgreSQL

Подключение

psql -d karusel

Просмотр таблиц

\dt

Адреса и ключи

SELECT id, name, api_key FROM locations;

Последние 20 событий

SELECT e.id, l.name AS адрес, m.name AS автомат, e.event_type, e.timestamp
FROM events e
JOIN locations l ON e.location_id = l.id
JOIN machines m ON e.machine_id = m.id
ORDER BY e.timestamp DESC LIMIT 20;

Очистка всех данных

DELETE FROM events;
DELETE FROM machines;
DELETE FROM jackpot_config;
DELETE FROM locations;

Полный сброс БД

dropdb karusel
