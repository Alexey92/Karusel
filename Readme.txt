Общая структура:
1. Адрес №N: игровые автоматы с ESP32 + локальный сервер. Локальная база данных - server/karusel.db. ESP32 отправляют события на локальный сервер. Тот записывает их в локальную базу данных и сразу пересылает на глобальный сервер. Информация на публичных экранах формируется из данных локальной базы данных.
2. Офис - глобальный сервер. Принимает данные от локальных серверов и отдельных ESP32 (по адресам, где один автомат).

Порядок администрирования серверов:
1. На локальном сервере во вкладке "Админ-панель" необходимо создать адрес и игровые автоматы. ID игровых автоматов необходимо проставить подряд начиная с 1. ESP32 для каждого игрового автомата необходимо прошить заменив в строке 22 прошивки MACHINE_ID на реальный адрес автомата. Адреса автоматов в "Админ-панели" должны соответствовать реальным адресам ESP32.
2. На глобальном сервере необходимо создать адрес, соответствующий реальному. Названия могут не совпадать - адреса сопоставляются исходя из ключа (будет описан ниже). Автоматы можно создавать и редактировать вручную, но по умолчанию они будут создаваться автоматически при поступлении события от локального сервера.


Windows (локальный сервер)

1. Автозапуск при включении ПК

2. Ручной запуск - открыть скрипт start_server.bat

3. Настроить start_server.bat для каждого конкретного адреса:
KARUSEL_CLOUD_URL - использовать ip адрес сервера
KARUSEL_CLOUD_KEY - см.вкладку "Адреса" на сервере, API Key
KARUSEL_LOCATION_ID - см.вкладку "Адреса" на сервере, ID

4. Открыть публичный экран - вставить в браузер http://192.168.1.100:5050/screen?monitor=0 или http://192.168.1.100:5050/screen?monitor=1 . Открыть 2 браузера, в каждом свой экран, разнести на разные мониторы.

5. Открыть админскую панель (конкретный адрес) - вставить  браузер http://localhost:5050/admin . Логин/пароль по умолчанию admin\admin123. Далее рекомендуется заменить пароль.

6. Очистка базы данных - удалить файл server\karusel.db

MAC OS (глобальный сервер)
 
1. Запуск сервера - открыть скрипт start_clous.sh

2. Открыть админ панель - localhost:5050/admin

3. Очистить БД - открыть скрипт reset_db.sh




Минимальный набор действий для запуска нового мини-ПК
1. Установка Python
Скачайте Python 3.12 с python.org. При установке обязательно отметьте галочку «Add Python to PATH». Проверьте: python --version.

2. Копирование проекта
Скопируйте папку karusel на мини-ПК (например, в C:\karusel). Можно с флешки.

3. Установка зависимостей Python
Откройте командную строку и выполните:

text
cd C:\karusel\server
python -m pip install -r requirements.txt
4. Настройка роутера на объекте
Подключитесь к роутеру, зайдите в 192.168.1.1. Задайте SSID karusel-net, пароль karusel2026. Закрепите IP мини-ПК как 192.168.1.100 через DHCP-резервацию.

5. Настройка start_server.bat
Откройте C:\karusel\start_server.bat в Блокноте. Пропишите:

text
set KARUSEL_CLOUD_URL=http://194.186.104.79:5050/api/event
set KARUSEL_CLOUD_KEY=ваш_api_key
set KARUSEL_LOCATION_ID=id_адреса
6. Автозапуск
Создайте ярлык для start_server.bat. Поместите его в shell:startup.

7. Первый запуск
Запустите start_server.bat. Откройте браузер: http://localhost:5050/admin. Войдите (admin/admin123). Создайте адрес и автоматы.
